# Kersul Dotfiles

This is my dotfiles config. One day it will have a installation script, for now it's just a backup repo :)

## Required Apps

- pip2/pip3
- ripgrep

## Commands that is need to be runned

### Install Plugins

Run inside neovim:

```
:PlugInstall
```

### Install Coc Eslint

Run inside neovim:

```
:CocInstall coc-eslint
```
